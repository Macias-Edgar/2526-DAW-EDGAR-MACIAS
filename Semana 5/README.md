# Galería Interactiva de Imágenes

Proyecto web desarrollado con HTML, CSS y JavaScript que permite crear una galería dinámica de imágenes usando URLs externas.

## Funciones principales
- Agregar imágenes ingresando una URL.
- Seleccionar imágenes con un clic.
- Resaltar solo una imagen a la vez.
- Eliminar la imagen seleccionada con botón o teclado.
